  - **삭제:** `외부 서비스 목킹` 섹션 전체(Supabase/Redis/OpenAI Mock 코드 블록 3개), `모범 사례` 10개 목록, `성공 지표`, `지속적 테스팅`, 중복 코드 예시

  - **유지:** frontmatter, TDD 7단계(압축), 테스트 유형(단위/통합/E2E), 파일 구조, 커버리지 80%, 주의사항

## 테스트 시나리오 (U1 완료 기준)

- 재작성 후 줄 수 ≤ 150줄
- `jest.mock(` 또는 `jest.fn(` 키워드 0개 (Mock 코드 블록 제거 확인)
- `통합 테스트 우선` 또는 `실 의존성` 문구 포함
- `Mock 허용 범위` 또는 `써드파티` 문구 포함
- frontmatter(`name`, `description`) 보존
- `TDD 7단계` 구조 유지
